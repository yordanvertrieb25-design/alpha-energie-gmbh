# Tech Context

## Technologies
- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (variables), Flexbox, Grid, Animations, Backdrop-filter for glassmorphism.
- **Vanilla JavaScript**: Used in `main.js` for interactive elements (e.g., scroll effects, mobile menu).
- **Python**: Utility scripts for image manipulation (`chroma.py`, `cutout.py`, `process_logo.py`, `smooth.py`).

## Development Environment
- OS: Windows
- Development relies on standard local editing, opening `index.html` in the browser, or using a local dev server.
- No heavy frameworks (React/Vue/Tailwind) are used, enforcing a lightweight vanilla approach.
