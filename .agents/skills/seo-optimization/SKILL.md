# SEO Optimization Instructions
- **Metadata**: Always include descriptive `<title>` tags and `<meta name="description">` tags for every page.
- **Headings**: Use a single `<h1>` per page. Follow a strict, logical heading hierarchy (`<h2>`, `<h3>`, etc.) without skipping levels.
- **Semantic HTML**: Use native semantic tags (`<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<footer>`, `<aside>`) instead of generic `<div>`s where applicable.
- **Images**: Always include meaningful `alt` attributes for images.
- **Links**: Ensure internal and external links use descriptive anchor text rather than "click here".
