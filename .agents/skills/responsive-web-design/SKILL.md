# Responsive Web Design Instructions
- **Mobile First**: Design layouts for mobile devices first, then use `min-width` media queries to enhance for tablets and desktops.
- **Modern Layouts**: Utilize CSS Grid and Flexbox for structural layout rather than floats or absolute positioning.
- **Fluid Sizing**: Use relative units (`rem`, `em`, `%`, `vw`, `vh`) over absolute units (`px`) for dimensions and typography.
- **Spacing**: Maintain consistent padding and margins across breakpoints to preserve visual rhythm.
- **Testing**: Ensure layouts do not break or require horizontal scrolling on narrow screens.
