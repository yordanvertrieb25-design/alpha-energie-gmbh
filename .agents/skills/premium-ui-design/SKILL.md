# Premium UI Design Instructions
- **Color Palette**: Avoid generic colors (plain red, blue, green). Use curated, harmonious color palettes, especially HSL tailored colors. Implement sleek dark modes where appropriate.
- **Typography**: Use modern Google Fonts (e.g., Inter, Roboto, Outfit). Ensure excellent readability and strong typographical hierarchy.
- **Visuals**: Use smooth gradients, glassmorphism, and subtle drop shadows to create depth.
- **Interactions**: Implement hover effects and micro-animations to make the interface feel responsive and alive.
- **Overall**: The interface must look extremely premium and state-of-the-art. Do not deliver simple minimum viable products when aesthetic excellence is expected.
